# Dyno frontend

React + TypeScript frontend, built with [Vite](https://vitejs.dev/). See the
repo root [CLAUDE.md](../CLAUDE.md) and [docs/](../docs/) for architecture,
API reference, and conventions.

## Scripts

- `npm start` — dev server on `http://localhost:3000`
- `npm run build` — production build to `build/`
- `npm run preview` — serve the production build locally
- `npm run test:e2e` — Playwright E2E suite (see root CLAUDE.md for details)
